# Controls

## By page in order of access

**Home (board selection):**
- `type` search
- `↑/↓` move
- `enter` select
- `esc/backspace` close search

**Catalog (thread list):**
- `h` home
- `↑/↓` move
- `enter` open thread
- `backspace` back

**Thread (post list):**
- `h` home
- `↑/↓` move
- `enter` open post / image
- `backspace` back
- `WASD` select link
- `e` open link

**Post view:**
- `h` home
- `backspace` back
- `WASD` select link
- `e` open link
- `enter` full image

**Full image view:**
- `h` home
- `backspace` back
